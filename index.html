Discussion Board Project


Project Background

The project was to pick a function of an existing LMS and improve on that function. We chose to work on the discussion board as it was lacking in many features that would make it much better for students and instructors.

Project Description

Our version of the project plans to reinvent the UI for the website, and we wanted to add functionality to the discussion board that made it more like a forum than an inbox.
